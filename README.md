# 🌏 관광-환율 연관 분석 대시보드  
한국관광공사의 관광 통계와 한국수출입은행 환율 정보를 기반으로,  
**관광객 수와 환율 간의 상관관계**를 분석하고 시각화한 Streamlit 대시보드입니다.

---

## 📌 프로젝트 주제  
관광객 수(출국∙방한)와 주요 국가 환율(USD, JPY, EUR, CNY) 간의  
**상관관계를 분석하고 대시보드 형태로 시각화**하는 프로젝트입니다.

---

## 🎯 주제 선택 이유  
- 환율 변동은 해외여행 및 방한 관광에 직접적인 영향을 미치는 중요한 요소이기 때문  
- 팬데믹 이후 한국인의 관광 회복 흐름과 환율 간 관계를 확인하고 싶었음  
- 실무형 데이터 분석: **데이터 수집 → 전처리 → EDA → 대시보드 개발 → 배포** 전 과정 경험  
- 팀 프로젝트로 협업 기반 데이터 분석 실습에 적합

---

## 📊 데이터 분석 내용

### 🔹 1) 데이터 수집  
- 한국관광공사 관광 통계 (CSV)  
- 한국수출입은행 환율 Open API

### 🔹 2) 데이터 전처리  
- 결측치 처리  
- 날짜/국가명 정규화  
- 월별·국가별 데이터 병합 (Pandas)

### 🔹 3) 주요 분석 기능  

#### 🏠 메인 대시보드  
- 최신 출입국 관광객 수 KPI  
- USD / JPY / EUR / CNY 환율 정보 표시  
- 월별 출입국자 수 비교 차트

#### 📈 환율–관광객 상관분석  
- 아웃바운드(출국) / 인바운드(방한) 선택  
- Pearson Correlation 계산  
- 관광객 수(막대) + 환율(선) 이중축 그래프 제공

#### 👨‍👩‍👧 성별∙연령대 분석  
- 성별·연령대 필터링  
- 월별 추이 및 연도별 히트맵 제공

#### 🛫 출국지별 분석  
- 인천·김해·제주 등 출국 공항/항만별 추이 분석

#### 🌏 방한 관광(인바운드) 분석  
- 성별·연령·방문 목적·교통수단별 월별 분석
